# Sudoku game

## Features

-   3 difficulty levels
-   Posibility to continue a previous game

## Configured with:

-   Webpack
-   ESLint
-   Prettier

## Installation

git clone https://github.com/angelynats/sudoku_app.git

cd sudoku-app

npm install / yarn install
